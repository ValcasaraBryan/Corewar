COMMANDES


Compiler un champion (creer un .cor a partir d'un .s):

./asm <champion.s>


Lancer le corewar avec le visu :

./corewar -n <champion.cor>


Check leaks :

while; do leaks corewar; sleep 1; clear; done;


Norminette :

norminette -R CheckForbiddenSourceHeader

------------------------

COMMITS


<type> [endroit] : <description>
	[informations supplementaires]


types :
	ajout
	progression
	correction
	documentation
	test
	amelioration
	norme

exemples :

ajout [parseur] : permet de lire les donnees des .cor
	lit les instructions live, xor, st et fork

test [asm] : creation de fichiers de test pour l'asm
	cf test1.s, test2.s, test3.s, etc

amelioration [vm] : optimisation
	les processus sont reorganises a chaque cycle pour plus d'efficacite

progression [vm] : ajout des couleurs
